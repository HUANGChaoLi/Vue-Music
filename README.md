# Vue-Music

> a music app built by Vue
> 一个简单的使用Vue构建的Web音乐播放器

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification and start server at port 3000
npm run serve

```
